Atualizando para uma nova versão do Lantern
===========================================

Quando lançarmos uma nova versão, mandaremos um email com um link para
baixar o instalador. Atualmente, mandar esse email é um processo manual.

Se você não receber o email, você pode usar o link no seu convite
original para baixar a versão mais recente.

.. figure:: https://dl.dropboxusercontent.com/u/253631/download_links.png
   :alt: Lantern invitation email

   Lantern invitation email
Nós estamos trabalhando em adicionar `notificações de
atualização <https://github.com/getlantern/lantern/issues/992>`__ no
Lantern. Por enquanto, os melhores lugares para acompanhar as novas
versões são:

-  `Twitter <https://twitter.com/getlantern>`__
-  `Tumblr <http://get-lantern.tumblr.com/>`__
-  `Facebook <https://www.facebook.com/getlantern>`__
-  `Google
   Groups <https://groups.google.com/forum/#!forum/lantern-users-en>`__
